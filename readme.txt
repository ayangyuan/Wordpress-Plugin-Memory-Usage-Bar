Show the current memory usage bar on admin header.

== Description ==

This plugin displays the current memory usage bar on the admin header.

You can easily see in the admin header the current memory usage, no more scoll down to the footer.

Plug, Activate, and Forget. 

A super lite plugin with out updating very now and them, simply does it job, and that's it.
